# CareFree - 无忧日记
## Description
- 一款用于记录日记以及便笺心情类的APP，你可以直观地看到你最近七天的心情变化，从而调节你的生活状态。
- 任何人可以fork代码自己编译运行, 但此项目仅供学习使用，禁止商用。
- 运行要求:  iOS 12.0 以上
- 开发环境:  Xcode 11
- 设计模式:  MVVM
